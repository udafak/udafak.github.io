First things first (structural elements)

Tag Name Description
<!DOCTYPE html> Document type Declaration It's used to define the document type and version of HTML being used. Always used at the beggining of documents
<html> HTML tag Encloses al of HTML code in the page Comes right after the doctype declaration
<head> Head tag Encloses the metadata of the pageLives inside the <html> tag
<body> Body tag Encloses the main content of the document, including text, images, links, and other elements that are visible to users. Lives inside the <html> tag
<link> Link element Used to link CSS files and web fonts Live inside the <head> element
